# StressNews Server

This is the backend API for the StressNews application. It provides news data and handles requests from the client app.

## Main App
- [StressNews Client (Live App)](https://stressnews-client-940cee89049b.herokuapp.com/)
- [Client Source Code](https://github.com/mpatrenin/nostressnews-client)

## Features
- RESTful API for news articles
- Tag and stress-level filtering
- Deployed on Heroku

## Getting Started
1. Install dependencies: `npm install`
2. Start server: `npm start`

## Deployment
- Deployed on Heroku
- See client README for deployment instructions
